Win probability formula:
1. Run projection: team wOBA → RPG using park factor + weather
2. Pythagorean expectation (exponent 1.83) from projected runs
3. Proprietary adjustment: actual home/away W% from 33k game data
4. Umpire bias: measured from historical game outcomes per umpire
5. Bullpen fatigue tax: -3% per reliever who pitched 2 consecutive days

Threshold (high confidence only):
• Minimum EV: 6% (fair win% minus market implied%)
• Minimum model win probability: 58%
• Both must be met — one alone is not enough
