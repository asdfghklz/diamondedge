Model Architecture
Data sources (100% real):
• MLB Stats API — official 2026 stats for every pitcher and team
• FIP: ((13×HR + 3×(BB+HBP) - 2×K) / IP) + 3.17
• wOBA: standard linear weights from counting stats
• FanDuel + BetMGM via The Odds API (free tier, ~2hr delay)
• OpenWeatherMap live weather per ballpark GPS
• 33,292 game retrosheet dataset: park factors, home/away WR, umpire bias

Odds note:
The free Odds API tier updates every 1-3 hours. Always verify the exact line on FanDuel or BetMGM before placing.

Win probability formula:
1. Team wOBA → runs per game (park + weather adjusted)
2. Pythagorean expectation (exponent 1.83)
3. Historical home/away split adjustment (33k games)
4. Umpire bias adjustment
5. Bullpen fatigue: -3% per reliever pitching 2 consecutive days

Thresholds:
• Minimum EV: 6% · Minimum win prob: 58% · Both required

Auto W/L:
• Engine fetches final scores at 11 PM CT and marks results
• Results stored in picks.json — syncs across all devices
• Manual override always available
